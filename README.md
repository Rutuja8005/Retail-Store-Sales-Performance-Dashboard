# 🏪 Retail Store Sales Performance Dashboard

## 📌 Project Overview

This project is a **Retail Sales Performance Dashboard** built using Power BI.
It provides a comprehensive view of sales, profit, and order performance across different regions, categories, and time periods.

---

## 📊 Key Metrics

* 💰 Total Sales: **₹ 718.93K**
* 📈 Total Profit: **₹ 91.95K**
* 📦 Total Orders: **1200**
* 📊 Profit Margin: **1278.95%**

---

## 📍 Features & Insights

* City-wise sales comparison (Top performing regions identified)
* Category-wise sales distribution:

  * Furniture
  * Office Supplies
  * Technology
* Monthly sales trend analysis
* Interactive filters:

  * Year filter (2021–2023)
  * Category filter
  * City filter
* Reset Filters button for quick dashboard reset

---

## 🛠 Tools Used

* Power BI
* Excel (Data Preparation)

## 📈 Key Insights

* United States generates the highest sales among all cities
* Sales peak observed around March–April
* Technology and Furniture categories contribute significantly to revenue
* Noticeable drop in sales during mid-year (July–September)

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Apply filters to explore different insights

## 📌 Conclusion

This dashboard helps businesses to:

* Track sales and profit performance
